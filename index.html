voii

Serverless Direct Chat

Send messages directly between devices via Apple Push Notifications. No servers, no accounts, no tracking.

How It Works

1
Share your identity Generate a QR code or deep link containing your device token
2
Send directly Messages are sent via APNs HTTP/2 — authenticated with ES256 JWT, no middleman
3
Receive instantly Background processing via Notification Service Extension saves messages before display

Features

💬

Direct Messaging

Send text messages instantly to any contact via push notifications

👥

Group Chat

Create groups with up to 5 members with client-side fan-out delivery

📝

Rich Markdown

Bold, italic, code blocks, lists, tables, checkboxes — full Markdown rendering

📋

Self Notes

Send messages to yourself as quick-access notes in your Inbox

☁️

iCloud Sync

Contacts and messages sync across all your Apple devices via CloudKit

📷

QR Code Sharing

Scan a QR code to instantly exchange identities and start chatting

🔒

Privacy First

No backend server, no accounts, no analytics — messages stay on your devices

📖

Open Source

Full source code available on GitHub — verify everything yourself
